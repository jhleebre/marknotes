# MarkNotes 테스트 가이드

MarkNotes의 모든 기능을 체계적으로 테스트하기 위한 가이드입니다.

## 📋 테스트 문서 목록

### 1️⃣ [텍스트 포맷팅 테스트](01.%20Text%20Formatting%20Test.md)

**테스트 항목:**
- H1~H6 헤딩 (6단계)
- 인라인 스타일 (볼드, 이탤릭, 취소선, 코드)
- 블록 요소 (블록쿼트, 수평선)
- 코드 블록 (언어별 하이라이팅)
- 단락과 줄바꿈

**예상 소요 시간:** 15분

### 2️⃣ [리스트 테스트](02.%20Lists%20Test.md)

**테스트 항목:**
- 불릿 리스트 (Cmd+Shift+8)
- 숫자 리스트 (Cmd+Shift+7)
- 중첩된 리스트 (5단계)
- 혼합 리스트 타입
- Tab/Shift+Tab 들여쓰기

**예상 소요 시간:** 20분

### 3️⃣ [테이블 테스트](03.%20Tables%20Test.md)

**테스트 항목:**
- 기본 테이블 생성
- 정렬 (왼쪽, 가운데, 오른쪽)
- Context menu 기능
- 테이블 내 복잡한 내용
- Tab 키 네비게이션

**예상 소요 시간:** 25분

### 4️⃣ [링크와 네비게이션 테스트](04.%20Links%20and%20Navigation%20Test.md)

**테스트 항목:**
- 외부 링크
- 내부 링크 (앵커)
- 링크 편집 (Cmd+K)
- 다양한 URL 형식
- Cmd+Click 동작

**예상 소요 시간:** 15분

### 5️⃣ [이미지 테스트](05.%20Images%20Test.md)

**테스트 항목:**
- 이미지 삽입 (JPG, PNG, GIF, SVG, WEBP)
- 크기 조절 (Small, Medium, Large, Original)
- Alt 텍스트 편집
- Base64 임베드
- .assets 폴더 관리

**예상 소요 시간:** 20분
**참고:** 실제 이미지 파일이 필요합니다.

### 6️⃣ [Task List 테스트](Task%20List%20Test.md)

**테스트 항목:**
- 체크박스 생성/토글 (Cmd+Shift+9)
- 중첩된 체크박스
- 테이블 내 체크박스
- 혼합 리스트 타입
- 상태 저장/로드

**예상 소요 시간:** 20분

### 7️⃣ [종합 테스트](06.%20Comprehensive%20Test.md)

**테스트 항목:**
- 실전 프로젝트 문서

- 블로그 포스트

**예상 소요 시간:** 30분

## ✅ 테스트 방법론

### 1. 문서 열기

파일 트리에서 테스트 문서를 클릭하여 엽니다.

### 2. Edit 모드 테스트 (Cmd+1)

- 포맷팅이 올바르게 표시되는지 확인
- 인터랙티브 요소 (체크박스, 링크) 클릭 테스트
- 편집 기능 테스트

### 3. Code 모드 테스트 (Cmd+2)

- 마크다운 문법이 올바른지 확인
- 프리뷰 패널에서 렌더링 확인
- 코드 편집 후 Edit 모드로 전환

### 4. 기능 테스트

각 문서의 "테스트 체크리스트" 섹션을 참고하여:
- [ ] 체크박스를 하나씩 확인하며 테스트
- [ ] 문제 발견 시 메모

### 5. 저장 및 재로드

- Cmd+S로 저장 (또는 자동 저장 대기)
- 파일을 닫았다가 다시 열기
- 내용이 보존되었는지 확인

### 6. 내보내기 테스트

- Cmd+Shift+P로 PDF 내보내기
- PDF에서 모든 요소가 포함되었는지 확인

## 💡 테스트 팁

### 효율적인 테스트

1. **하나씩 순차적으로**
   - 여러 문서를 동시에 열지 말고 순서대로 진행

2. **체크리스트 활용**
   - 각 문서의 테스트 체크리스트를 반드시 확인

3. **모드 전환 반복**
   - Edit ↔ Code 모드를 자주 전환하며 테스트

4. **Edge Case 테스트**
   - 극단적인 케이스도 시도해보기
   - 매우 긴 텍스트, 많은 중첩 등

### 주의사항

⚠️ **이미지 테스트**
- 실제 이미지 파일이 필요하므로 미리 준비

⚠️ **파일 저장**
- 테스트 중 자동 저장이 활성화되어 있음
- 원본을 보존하려면 복사본 생성

⚠️ **성능 테스트**
- 큰 문서(많은 이미지, 테이블)는 성능에 영향을 줄 수 있음

## 🔧 문제 해결

### 자주 발생하는 이슈

#### 1. 체크박스가 표시되지 않음

**해결방법:**
- Code 모드에서 `- [ ]` 문법 확인
- Edit 모드로 전환
- 페이지 새로고침 (파일 재로드)

#### 2. 테이블 정렬이 안 됨

**해결방법:**
- 구분자 행 확인: `| :--- | :---: | ---: |`
- Context menu에서 정렬 재설정

#### 3. 링크가 작동하지 않음

**해결방법:**
- Edit 모드에서 Cmd+Click 사용
- URL 형식 확인 (`https://` 포함)

#### 4. 이미지가 표시되지 않음

**해결방법:**
- `.assets` 폴더 확인
- 이미지 경로 확인
- 지원 형식 확인 (JPG, PNG, GIF, SVG, WEBP)
