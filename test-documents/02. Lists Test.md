# 리스트 테스트 문서

이 문서는 모든 리스트 타입과 중첩, 들여쓰기 기능을 테스트합니다.

## 불릿 리스트 (Cmd+Shift+8)

### 단순 불릿 리스트

- 첫 번째 항목
- 두 번째 항목
- 세 번째 항목
- 네 번째 항목

### 불릿 리스트 + 텍스트 포맷팅

- **굵은 텍스트** 항목
- _이탤릭 텍스트_ 항목
- ~~취소선~~ 항목
- `코드` 항목
- 일반 텍스트와 **볼드**와 *이탤릭*을 섞은 항목

---

## 숫자 리스트 (Cmd+Shift+7)

### 단순 숫자 리스트

1. 첫 번째 항목
2. 두 번째 항목
3. 세 번째 항목
4. 네 번째 항목
5. 다섯 번째 항목

### 숫자 리스트 + 텍스트 포맷팅

1. **중요한** 첫 번째 단계
2. _강조된_ 두 번째 단계
3. `코드`를 실행하는 세 번째 단계
4. 일반적인 네 번째 단계

---

## 중첩된 불릿 리스트 (Tab/Shift+Tab)

### 2단계 중첩

- 최상위 항목 1
  - 하위 항목 1-1
  - 하위 항목 1-2
  - 하위 항목 1-3
- 최상위 항목 2
  - 하위 항목 2-1
  - 하위 항목 2-2

### 3단계 중첩 (마커 스타일 변경 확인)

- 레벨 1 (disc ●)
  - 레벨 2 (circle ○)
    - 레벨 3 (square ■)
      - 레벨 4 (▫)
        - 레벨 5 (⁃)

### 복잡한 중첩 구조

- 프로젝트 A
  - 단계 1: 기획
    - 요구사항 정의
    - 시장 조사
    - 경쟁사 분석
  - 단계 2: 설계
    - UI/UX 설계
    - 데이터베이스 설계
    - API 설계
  - 단계 3: 개발
    - 프론트엔드
      - React 컴포넌트
      - 상태 관리
    - 백엔드
      - REST API
      - 데이터베이스 연동

## 중첩된 숫자 리스트

### 2단계 중첩

1. 첫 번째 메인 항목
   1. 하위 항목 1
   2. 하위 항목 2
   3. 하위 항목 3
2. 두 번째 메인 항목
   1. 하위 항목 1
   2. 하위 항목 2

### 3단계 중첩

1. 단계 1
   1. 세부 단계 1-1
      1. 상세 1-1-1
      2. 상세 1-1-2
   2. 세부 단계 1-2
      1. 상세 1-2-1
      2. 상세 1-2-2
2. 단계 2
   1. 세부 단계 2-1
   2. 세부 단계 2-2

## 긴 내용이 있는 리스트

### 여러 줄 항목

- 이것은 매우 긴 리스트 항목입니다. 이 항목은 여러 줄에 걸쳐서 표시되며, 텍스트가 자동으로 줄바꿈됩니다. 두 번째 줄도 올바르게 들여쓰기가 되어야 합니다.

- 또 다른 긴 항목입니다. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam.
  - 중첩된 긴 항목도 테스트합니다. 이 항목 역시 여러 줄에 걸쳐 표시될 수 있으며, 올바른 들여쓰기를 유지해야 합니다.

### 리스트 내 코드 블록

- 설치 방법:

  ```bash
  npm install
  npm run dev
  ```

- 실행 방법:
  ```bash
  npm start
  ```

### 첫 번째 항목 테스트

  - 들여쓰기가 안 되어야 함 (첫 항목이므로)

- 두 번째 항목에서 Tab을 누르면?
  - 정상적으로 들여쓰기됨
