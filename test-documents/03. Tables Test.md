## Tab 키 네비게이션 테스트

### 일반 셀 네비게이션

다음 테이블에서 Tab/Shift+Tab 테스트:

| A | B | C |
| :--- | :--- | :--- |
| 1 | 2 | 3 |
| 4 | 5 | 6 |

**테스트 시나리오:**
1. 첫 번째 셀에 커서 위치
2. Tab 키: 다음 셀로 이동
3. 마지막 셀에서 Tab: 새 행 생성
4. Shift+Tab: 이전 셀로 이동

### 리스트가 있는 셀에서 Tab 테스트

| 작업 | 세부사항 |
| :--- | :--- |
| 개발 | - 기능 A<br>- 기능 B |
| 테스트 | - 단위 테스트<br>- 통합 테스트 |

셀 안의 리스트에서 Tab을 누르면 리스트가 들여쓰기 되어야 합니다.
