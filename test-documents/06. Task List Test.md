## 특수 케이스 테스트

### 긴 텍스트와 함께

- [ ] 이것은 매우 긴 텍스트를 가진 체크박스 항목입니다. 이 항목은 여러 줄에 걸쳐서 표시될 수 있으며, 텍스트가 길어도 체크박스가 제대로 정렬되는지 확인하기 위한 테스트입니다.
- [x] 완료된 긴 항목도 마찬가지로 여러 줄에 걸쳐 표시될 수 있습니다. 취소선이 모든 텍스트에 적용되는지 확인해야 합니다.

### 블록쿼트와 함께

> - [ ] 인용구 안의 체크박스 1
> - [x] 인용구 안의 체크박스 2
>   - [ ] 중첩된 체크박스

### 코드 블록 근처

```markdown
- [ ] 이것은 코드 블록 안의 예시입니다
- [x] 실제로 렌더링되지 않습니다
```

위는 코드 블록이고, 아래는 실제 체크박스:

- [ ] 코드 블록 후의 실제 체크박스
- [x] 완료된 실제 체크박스
