## 링크 클릭 동작 테스트

### Cmd+Click으로 링크 열기

Edit 모드에서 다음 링크들을 Cmd+Click하여 테스트:

- [Google 열기](https://google.com) - 새 브라우저 창
- [GitHub 열기](https://github.com) - 새 브라우저 창
- [내부 링크](#목차-내부-링크-테스트) - 스크롤 이동

### 일반 클릭 동작

Edit 모드에서 링크를 일반 클릭하면:

- 링크가 활성화되지 않음 (Cmd+Click 필요)
- 커서가 링크 위치에 놓임
