# Welcome to MarkNotes

A **beautiful**, *distraction-free* markdown editor for your thoughts and ideas.

> Write in markdown, see it rendered instantly. No cloud, no accounts - just your notes, stored locally as plain `.md` files.

## Features at a Glance

| Feature | Description |
| :--- | :--- |
| WYSIWYG Editing | Rich text editing with live preview |
| Syntax Highlighting | Code blocs in 30+ languages |
| Advanced Tables | <ul><li><p>Header, body, alignment support</p><ul><li><p>Nested lists inside cells</p></li></ul></li><li><p>Flexible column layout</p></li></ul> |

## Quick Example

```
const greeting = (name: string): string => {
	return `Hello, ${name}! Welcome to Marknotes.`
}
console.log(greeting('World'))
```

## Your Notes, Your Way

- [x] Local-first - all files in `~/Documents/MarkNotes/`
